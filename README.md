# 📚 Gerenciador de Biblioteca

Este é um projeto Full Stack desenvolvido para gerenciar o acervo de uma biblioteca pessoal, permitindo as operações de criar, listar, editar e excluir livros.

### 🛠 Tecnologias Utilizadas:
* **Frontend:** HTML5 Semântico, CSS3 e JavaScript.
* **Backend:** Node.js com Express.
* **Banco de Dados:** MySQL.

### 🚀 Como rodar o projeto:
1. Clone o repositório.
2. Importe o banco de dados usando o script SQL.
3. No terminal, rode `npm install` para instalar as dependências.
4. Inicie o servidor com `node server.js`.
5. Abra o arquivo `index.html` no navegador.
